# お酒とさかな かけはし LP

大阪・福島の和食店「お酒とさかな かけはし」のスマホ優先ランディングページです。

## 構成

- `index.html`：LP本体
- `styles.css`：和モダンのレスポンシブデザイン
- `script.js`：画像スライダー、浮かび上がる表示
- `assets/`：店舗提供・公式掲載許諾素材
